## Laravel-8 Authorization

A Simple Laravel app to perform authorization process.

Package required : Laravel ui for authentication

First of all, $git clone    
open the directory
run $composer install
$ cp .env.example .env
$php artisan key:generate
create a database
run $php artisan migrate
run $composer require laravel/ui
run $php artisan ui vue --auth (vue for the frontend template-optional)
run $php artisan serve
